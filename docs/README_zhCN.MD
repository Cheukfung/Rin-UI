## 📄 介绍

RinUI 是一个类 Fluent Design 的 Qt Quick (QML) UI 库，基于 PySide6，使其中的 QML 组件可以和 Python 进行通信，如主题系统。本UI库提供了一些高质量的 Fluent 风格控件，可用于您的 QML 应用程序中。

## 🪄 使用方法

你可以通过 pip 安装 RinUI：#可能不会（coming soon）
```bash
pip install RinUI
```
在你的 QML 文件中导入 RinUI：

```qmllang
import "./RinUI"
```

你还可以在源码中查看示例，如下所示：

```bash
cd examples
python gallery.py
```

你也可以将 RinUI 文件夹移动到你的项目目录中的任意位置。
